DATA TYPES

  • Numeric Data Types:
    Integer: Used to store whole numbers (e.g., INT, INTEGER, SMALLINT, BIGINT).
    Floating-Point: Used to store decimal numbers (e.g., FLOAT, DOUBLE, REAL).
    Decimal/Numeric: Used for precise decimal values (e.g., DECIMAL, NUMERIC)..
  • Character Data Types:
    Character String: Used to store fixed-length strings (e.g., CHAR, NCHAR).
    Variable-Length String: Used to store strings of varying lengths (e.g., VARCHAR, NVARCHAR).
    Text: Used to store large text or character data (e.g., TEXT, CLOB).
  • Date and Time Data Types:
    Date: Used to store dates (e.g., DATE, DATETIME, TIMESTAMP).
    Time: Used to store time values (e.g., TIME, TIME WITH TIME ZONE).
    Interval: Used to store time intervals or durations (e.g., INTERVAL).
